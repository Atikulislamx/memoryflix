# 💙 About

MemoryFlix is a personal cinematic memory website inspired by the storytelling experience of Netflix.

Instead of watching movies, visitors explore memories.

Every photograph becomes a scene.

Every diary page becomes a chapter.

Every video becomes an episode.

Every conversation becomes part of an unforgettable story.

This project is being built as a living digital archive that grows over time with new memories.
